# Examples

This directory contains example outputs and usage demonstrations.

## Example Output Format

Outputs are typically stored in JSON format for easy parsing and analysis:

```json
{
  "model": "gpt-3.5-turbo",
  "prompt": "Explain retrieval-augmented generation",
  "output": "Retrieval-Augmented Generation (RAG) is a technique...",
  "timestamp": "2025-12-08T10:54:00Z",
  "metadata": {
    "temperature": 0.7,
    "max_tokens": 500
  }
}
```
